
# setup oh-my-posh with my custom theme


## Windows

 * Requirements
   * PowerShell >= 5
   * winget
 * Setup Script: `setup_win.ps1`
 * Uninstall Script: `uninstall_win.ps1`

## Debian/Ubuntu

 * Requirements
   * bash
   * wget
   * unzip
   * fontconfig
 * Setup Script: `setup_debian_ubuntu.sh`
 * Uninstall Script: `uninstall_debian_ubuntu.sh`

## oh-my-posh

* Version: `>= 12.22.0`
